About Me

Hi there! I'm Bo Zhong, a full-stack software engineer based in the beautiful San Francisco Bay Area. I love turning ideas into reality by leveraging my expertise in a variety of programming languages, including JavaScript, Ruby on Rails, Swift, and React. When I'm not coding away, I enjoy exploring the latest trends in Firebase Auth, Cocoapods, and Swift Package Manager. I'm also a bit of a whiz with RESTful API calls and delegate design pattern implementation.

Some of the coolest projects I've worked on include Mighty Ninja Run, Quizzler, and Clim8 - a trio of awesome iOS apps that showcase my skills in Swift, MVC design patterns, and principles of object immutability. My other claim to fame is Home-BnB, a full-stack Airbnb clone built using Ruby on Rails and React.js. I'm also super proud of Project Sticky, a MERN-based work management app that's all about helping people track and manage their work with an intuitive UI.

Fullstack Projects

an online marketplace focused on short-term homestay and experiences. homebnb is a full-stack, multi-functional application clone of Airbnb that uses React and Redux on the front-end and Ruby on Rails, PostgresSQL and AWS S3 on the back-end.

Project Sticky - Task Managing App

Project Sticky is an app inspired by the need for project management. By the name project sticky, we hope people can stick to the plan and finish their projects on time. A virtual representation of the progress will make our app more entertaining and practical for people to notice where they are at in the project timeline, as well as if they are on track or behind.
